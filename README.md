# makeathon-pong
Code for the Commander Keen Pong game we are making for the Maker Effect Foundation Makeathon 0.0

# Board
Arduino UNO

# Pin Out

## Ultrasonic Sensor 1
- Echo: Digital 2
- Trig: Digital 3

## Ultrasonic Sensor 2
- Echo: Digital 7
- Trig: Digital 6

## DC Volt Analog Meter
- Input: Digital ~11
- Output: GND
